# 🚀 AI Investment Quant Dashboard (Enterprise v2.0)

完全自律型のAIヘッジファンド運用システムへようこそ！
本システムは、米国の「Alpaca証券」のAPIと連携し、相場の分析から発注、リスク管理、そして損益のSlack報告まですべてを全自動で行うプロ仕様のクオンツ・トレード・ボットです。

---

## 🛠️ システムアーキテクチャ（2段階ジョブ構造）

システムはクラウド（GitHub Actions）上で、毎日決まった時間に自動起動します。

1. **Job 1: Trade Bot (午前04:00 JST / 米国市場クローズ1時間前)**
   - 米国市場が開いている最後の1時間に起動します（最も出来高が多く、日足の形が確定するゴールデンタイム）。
   - 監視対象銘柄の過去1年分のデータを取得し、AIがテクニカル分析（MACD, RSI）を行います。
   - 買いシグナルが点灯すれば、自動で「指値注文」を市場に流し込みます。

2. **Job 2: Summary Bot (午前06:30 JST / 米国市場クローズ後)**
   - 市場が完全に閉まった後に起動し、当日の注文が約定したかを最終確認します。
   - 口座全体の総資産、前日比、保有銘柄の個別の収益率を計算し、Slackへサマリーを報告します。
   - 新しく購入した（保有している）銘柄すべてに対し、**ボラティリティ（ATR）連動型のトレイリングストップ（損切りライン）を自動で設定**し、翌日の暴落に備えます。

## 📈 投資ルール（売買ロジック）

AIは感情を完全に排除し、以下の数学的・統計的ルールのみに従ってトレードを行います。

### 【買】エントリー条件
以下のいずれかの条件を満たした場合にのみ買い注文を出します。
*   **MACD ゴールデンクロス**: 中期的な上昇トレンドへの転換を確認。
*   **大底反発 (RSI 30以下からの脱出)**: 売られすぎ水準から反発し、明確な底打ちを確認。
*   ※プレマーケット暴落フィルター: 条件を満たしていても、時間外取引で「前日比 -3%以上」の下落があれば、悪材料が出たとみなして発注を直前でキャンセルします。

### 【売】エグジット条件
以下のいずれかの条件を満たした場合、保有する全株を即座に「成行」で売却し、現金へ退避します。
*   **MACD デッドクロス**: 上昇トレンドが崩れ、下落トレンドへの転換を確認。
*   **超過熱 (RSI 75以上からの下落)**: 買われすぎ水準から失速し、天井を打ったと判断。
*   **自動防衛線（トレイリングストップ）への接触**: 各銘柄のボラティリティ（値動きの幅）に応じて自動設定された損切りラインを割った場合、証券会社のシステム側で自動的に売却されます。

## 🛡️ 鉄壁のリスク管理システム

大切な資産を守るため、機関投資家レベルの4重の防衛網を敷いています。

1. **動的ポジションサイジング (ダイナミックロット)**
   資金の全額を1つの銘柄に突っ込むことは決してしません。各銘柄の値動きの激しさ（ATR）を計算し、「最悪のケースでも、総資産の2%しかダメージを受けない」ように、購入する株数を自動で逆算・調整します。
2. **ATR連動型トレイリングストップ**
   保有銘柄に対して、値動きの激しい銘柄は広く（例：8%）、安定した銘柄は狭く（例：3%）、最適な損切り幅をAIが毎日自動計算して証券会社に設定します。
3. **プレマーケット暴落フィルター**
   「チャート上は買い」でも「ニュースで悪材料が出て時間外で暴落している」というギャップを防ぎます。
4. **キルスイッチ（絶対防衛ライン）**
   もし市場全体がパニックになり、ポートフォリオ全体で「-5%」のドローダウン（資産減少）を検知した場合、強制的にすべての新規購入プログラムをロックダウン（凍結）し、暴落の直撃を防ぎます。

## 🎮 ユーザーの操作・運用ルール

本システムは完全自動（ほったらかし）で動作しますが、**あなたが操作すべきファイルはたった1つだけ**です。

### 監視銘柄（ウォッチリスト）を変更したい場合
`config.py` ファイルを開き、`TARGET_SYMBOLS` のリストを書き換えるだけです。

```python
TARGET_SYMBOLS = [
    "MSFT",  # マイクロソフト
    "NVDA",  # エヌビディア
    "AAPL",  # アップル
    "BTC-USD" # ビットコインも可能！
]
```

リストを編集して保存（GitHubへPush）すれば、明日の朝からAIが自動的に新しい銘柄の監視と分析を開始します。

### その他
*   初期設定のパラメータ（リスク許容度2%など）を変更したい場合も、すべて `config.py` 内の数値をいじるだけでシステム全体に適用されます。
*   手動で好きな株を買った場合でも、翌朝のシステムが検知して自動的に防衛線を張ってくれます。
